Localiser les logs par services

Une erreur ? Un accès refusé ? C'est dans les logs que vous trouverez des indices ! Cependant.. Il faut déjà savoir où les trouver ces fameux logs.

 

Horizon (dashboard)

Pour le dashboard Horizon, les logs se trouve sur les controllers.

Pour les logs de connexions :

/var/log/apache2/horizon_ssl_error.log
